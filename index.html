    What is this?

    Listen to PyPI turns the Python Package Index into a live ambient soundscape. Every time a package is published or updated, you hear it as a tone and see it as a glowing circle.

    What you'll hear

    • A gentle background drone plays continuously - a warm chord that creates a meditative foundation (toggle it off with the Drone switch in the header)
    • Bell chimes ring for minor and patch releases, with pitch and timbre varying by package category
    • Deep singing bowl tones mark new packages and major version releases
    • Occasional shimmers add textural variety

    What you'll see

    • Gold circles for brand-new packages
    • Blue circles for major releases
    • Teal circles for minor updates
    • Gray circles for patch fixes

    Leave it running in a background tab for a calm, ever-changing soundscape driven by the Python community's real-time activity.

    Privacy

    This site only reads public PyPI RSS feeds. It doesn't send or store any data - the only things saved are your volume and drone settings in localStorage.
